## ▶️ Getting started

**1. Clone the project**

```
git clone https://github.com/RodolfoSilveira/aguas-front.git
```

**2. Install dependencies**

```
yarn
or
npm install
```

**3. Run Test**
`yarn test`
or
`npm run test`

**4. Run APP**

`yarn start`
or
`npm start`
